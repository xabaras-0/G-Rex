= G-Rex
:warning-caption: :warning:

G-Rex is a simple OSINT tool to automate scraping of publicly available mail addresses for a specific domain.
Data is searched through Google Search, specifiyng domain and number of Google pages to scroll (example: `grex.py gmail.com 2`).

WARNING: Extensive use of this script may result in a temporary ban of automatic requests to Google from your IP.

----
$ grex.py --help

usage: grex.py [-h] domain pages

positional arguments:
  domain
  pages       number of pages to scrape

optional arguments:
  -h, --help  show this help message and exit

----


example:
----
$ python3 grex.py  gmail.com 2
